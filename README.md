# week-api

I created this repository for example web api example.
This API calculate counter weeks from any start date to current date. By week start on monday.
Reture a result in format 
> total week counts + white space + Current year + '-W' + Number of weeks in year.


Here is week start on monday standard description I found on google.
>According to the
>ISO 8601 international standard, Monday is designated as the first day of the week, often used in business, finance, and most countries in Europe, Asia, and Oceania. This convention aims to keep the work week >(Monday–Friday) together and treats Saturday and Sunday as the closing "weekend". 
>
>   * ISO 8601 Standard: Defines Monday as Day 1 and Sunday as Day 7.
>   * International Use: Predominant in Europe, South America, Africa, and Asia, as well as in Australia and New Zealand.
>    Contrast with Sunday Start: Many countries, including the USA, Canada, Japan, and parts of Latin America, consider Sunday the first day of the week, often based on religious or traditional calendars.
>   * Business Practice: Monday start is widely used in business systems and digital calendars (e.g., ISO weeks) to avoid splitting the weekend.
